# GridReady AI Logo Assets

Canonical color: `#1B365D`

## Structure

- `source/logo.svg`: primary editable SVG source
- `vector/logo.pdf`: vector PDF export
- `raster/`: general transparent PNG exports
- `web/`: favicon, app icons, and web manifest files

The web folder keeps conventionally named browser assets. The raster folder keeps only general-purpose PNG sizes to avoid duplicate copies of the same export.
